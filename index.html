Mappa di apprendimento HTML

ARGOMENTI BASE DA CONOSCERE IN HTML

Clicca sulle sezioni per espandere i dettagli. Segui l'ordine consigliato: dalla radice alle foglie.

1 Radice — Basi di HTML Fondamenta
  • Struttura pagina — <!DOCTYPE html>, <html>, <head>, <body>
  • Tag fondamentali — <h1>...<h6>, <p>, <br>, <hr>
  • Commenti — <!-- commento -->
2 Testo e contenuti Markup del contenuto
  • Formattazione — <strong>, <em>, <b>, <i>
  • Liste — <ul>, <ol>, <li>
  • Link — <a href="..."> e attributi rel, target
3 Media Immagini, audio, video
  • <img> — src, alt, width, height (accessibilità)
  • <audio> & <video> — attributi controls, preload, loop
  • <iframe> — incorporamento (attenzione alla sicurezza e al sandbox)
4 Struttura semantica (HTML5) Buone pratiche
  • <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>
  • <div> vs <span> — quando usare elementi non semantici
5 Tabelle Dati tabellari
  • <table>, <tr>, <td>, <th> — struttura e accessibilità
  • Attributi utili — rowspan, colspan
6 Form e input Interazione utente
  • <form> — action, method (GET/POST)
  • Input: <input type="text|email|password|file|checkbox|radio">
  • <label>, <select>, <textarea>, <button>
  • Validazione HTML5 (required, pattern, type=email)
7 Attributi globali & Accessibilità Attraversabilità e SEO
  • id, class, title, lang, hidden
  • Attributi ARIA essenziali (aria-label, role)
  • alt per immagini e testo alternativo
8 Argomenti avanzati Meta, integrazione e best practice
  • Meta tag: charset, viewport, description (SEO)
  • Link a CSS (<link rel="stylesheet">) e script (<script src="...">)
  • Responsive basics e immagini srcset
  • Progressive enhancement, performance e sicurezza (CSP, sanitizzazione)
Suggerimento rapido: studia esercitandoti con mini-progetti (pagina personale, form di contatto, galleria immagini). Il modo tradizionale di imparare è provare e correggere — continua così!
CSS

Mappa di apprendimento CSS

Percorso ordinato per imparare i concetti principali di CSS, dai fondamenti alle tecniche più avanzate.

Clicca sulle sezioni per espandere i dettagli. Segui l'ordine consigliato per costruire una base solida.

1 Fondamenti CSS Basi
  • Collegare CSS all'HTML: <link>, inline, internal
  • Sintassi base: selettore { proprietà: valore; }
  • Unità di misura: px, %, em, rem, vh, vw
2 Selettori Targeting degli elementi
  • Selettori di tipo, classe, ID
  • Selettori combinatori: discendente, figlio, fratello
  • Pseudo-classi: :hover, :focus, :nth-child()
  • Pseudo-elementi: ::before, ::after
3 Box Model Struttura di ogni elemento
  • Contenuto, padding, bordo, margine
  • box-sizing: content-box vs border-box
4 Colori, sfondi e testo Stile base
  • Colori: nome, hex, rgb, hsl
  • Background: colore, immagine, posizione, repeat
  • Tipografia: font-family, font-size, line-height, text-align
5 Layout Disposizione degli elementi
  • Display: block, inline, inline-block, none
  • Position: static, relative, absolute, fixed, sticky
  • Flexbox: container e item, justify-content, align-items
  • Grid: righe, colonne, area
  • Float e clear (metodi tradizionali)
6 Transizioni e animazioni Dinamismo
  • Transizioni: transition-property, duration, timing-function
  • Transform: translate, rotate, scale
  • Animazioni con @keyframes
7 Responsive Design Adattabilità
  • Media queries
  • Unità relative (%, em, rem, vh, vw)
  • Layout fluidi e immagini responsive
8 Avanzato Tecniche moderne
  • Variabili CSS (custom properties)
  • Funzioni utili: calc(), clamp(), min(), max()
  • CSS avanzato: clip-path, filter, backdrop-filter
  • Gestione temi (light/dark mode)
Consiglio: costruisci piccoli progetti pratici come card, navbar o layout di blog. La pratica costante è il modo più solido per padroneggiare i CSS.

JavaScript

Mappa di apprendimento JavaScript

Percorso ordinato per imparare i concetti principali di JavaScript, dai fondamenti alle tecniche avanzate.

Espandi le sezioni per consultare i vari argomenti consigliati. Segui l'ordine per una comprensione progressiva.

1 Fondamenti Le basi del linguaggio
  • Storia e ruolo di JavaScript
  • Includere JS in HTML: <script>
  • Console e debugging base
  • Sintassi, variabili (let, const, var)
  • Tipi di dato: string, number, boolean, null, undefined, object
2 Operatori e strutture di controllo Logica di base
  • Operatori aritmetici, logici e di confronto
  • if/else, switch
  • Loop: for, while, do...while
3 Funzioni Riutilizzare codice
  • Dichiarazione e chiamata di funzioni
  • Parametri e valori di ritorno
  • Funzioni anonime ed arrow functions
  • Scope e closure
4 Oggetti e array Strutture dati
  • Creazione e proprietà degli oggetti
  • Metodi e this
  • Array: metodi principali (push, pop, map, filter, reduce)
  • Iterazioni sugli array
5 DOM e Eventi Interazione con la pagina
  • Selezione elementi: getElementById, querySelector
  • Modifica contenuto e stile
  • Gestione eventi: addEventListener
  • Event bubbling e delegation
6 Programmazione asincrona Gestione del tempo
  • setTimeout, setInterval
  • Callback
  • Promises
  • Async/Await
  • Fetch API (richieste HTTP)
7 Concetti avanzati Approfondimenti
  • ES6+ features (destructuring, spread, modules)
  • Gestione errori con try/catch
  • JSON e localStorage
  • Classi e prototipi
  • Moduli JavaScript (import/export)
8 Strumenti e buone pratiche Per lavorare meglio
  • Console avanzata e debugging con DevTools
  • Linting e formattazione (ESLint, Prettier)
  • GitHub e versioning
  • NPM e gestione pacchetti
  • Framework e librerie (React, Vue, Angular — introduzione)
Suggerimento: esercitati con piccoli progetti come un to-do list, una calcolatrice o una galleria di immagini interattiva. La pratica costante è la chiave per padroneggiare JavaScript.
